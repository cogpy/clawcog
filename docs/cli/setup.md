# `opencog setup`

Initialize `~/.opencog/opencog.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
opencog setup
opencog setup --workspace ~/.opencog/workspace
```

To run the wizard via setup:

```bash
opencog setup --wizard
```
