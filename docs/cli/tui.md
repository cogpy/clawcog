# `opencog tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
opencog tui
opencog tui --url ws://127.0.0.1:18789 --token <token>
opencog tui --session main --deliver
```
